## 🚀 About the Project

This is the source code for my personal professional portfolio. It is designed to be a robust showcase of my skills as a Final-year CS student, Full-Stack Developer, and AI Researcher. Rather than relying on heavy front-end frameworks, this portfolio is built entirely using **Vanilla Web Technologies**, focusing on high performance, raw DOM manipulation, custom animations, and a unique user experience.

### 🔥 Key Features
- **3D & Interactive Elements:** Parallax scrolling backgrounds, floating gradient orbs, and premium custom hover states.
- **AI Chatbot Assistant:** A custom-built, simulated AI widget that can answer contextual questions about my resume, skills, and past projects.
- **Developer Terminal Easter Egg:** Press `Ctrl + ~` to open a fully functional, simulated Linux terminal to explore hidden commands, run 'projects', and discover easter eggs.
- **Premium UI/UX:** Warm, sophisticated color palette, dynamic typography scaling, glassmorphism effects, and highly responsive design across all devices.
- **Serverless Contact Form:** Active contact form linked directly to my inbox via FormSubmit, completely eliminating the need for a backend server.

## 🛠️ Technology Stack
* **HTML5** (Semantic structure, accessibility)
* **CSS3** (Custom properties, grid, flexbox, keyframe animations, media queries)
* **Vanilla JavaScript** (DOM manipulation, event handling, chatbot conversational logic, terminal command parsing)

## 💻 Running Locally
Since this project deliberately avoids complex build tools or `node_modules`, running it locally is incredibly simple:

1. Clone the repository:
   ```bash
   git clone https://github.com/Neerav02/Neerav_Babel.git
   ```
2. Navigate to the directory:
   ```bash
   cd Neerav_Babel
   ```
3. Double-click `index.html` to open it in any modern web browser.
*(Alternatively, use an extension like VS Code Live Server for hot-reloading during development).*
